# СОСТОЯНИЕ 0: Инициализация

**Статическая память: R: [0,0,0] P: [?, ?, ?] Стэк: [ main ]**

Точка входа в программу. На этом этапе определяются константы, выделяется память под массивы P (сама перестановка) и R (флаги занятых чисел). Массив R заполняется нулями.

<pre>
#include &lt;iostream&gt;
using namespace std;

int main() {
    const int n = 3;
    int P[n];
<b>---> int R[n] = {0}; </b>

    for (int i1 = 0; i1 < n; i1++) {
        P[0] = i1;
        R[i1] = 1;
        for (int i2 = 0; i2 < n; i2++) {
            if (R[i2] == 0) {
                P[1] = i2;
                R[i2] = 1;
                for (int i3 = 0; i3 < n; i3++) {
                    if (R[i3] == 0) {
                        P[2] = i3;
                        R[i3] = 1;
                        cout << P[0] << " " << P[1] << " " << P[2] << endl;
                        R[i3] = 0;
                    }
                }
                R[i2] = 0;
            }
        }
        R[i1] = 0;
    }
    return 0;
}
</pre>

# СОСТОЯНИЕ 3: i3 = 2 (ВЫВОД №1)

**Статическая память: R: [1,1,1] P: [0, 1, 2] Стэк: [ main → i1=0 → i2=1 → i3=2 ]**

Третий цикл находит свободное число 2. Оно записывается в `P[2]`. Выводится первая перестановка: 0 1 2. Сразу после этого — **бэктрекинг** для i3.

<pre>

                    if (R[i3] == 0) { 
                        P[2] = i3; 
                        R[i3] = 1; 
<b>--->                    cout << P[0] << " " << P[1] << " " << P[2] << endl; </b>

**Бэктрекинг** в действии: индекс 1 был освобожден (`R[i2]=0`), второй цикл перешел к `i2=2`. Число 2 записывается в `P[1]`.

<pre>

            }

<b>--->    if (R[i2] == 0) { P[1] = i2; R[i2] = 1; } // i2=2 </b>
